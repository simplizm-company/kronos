jQuery datepicker plugin - Kronos

1. 개요

데이트피커 플러그인 'Kronos' 입니다.
날짜선택, 기간설정, 콜백함수등을 제공하고 있으며
사용자피드백을 통해 지속적으로 업데이트 할 예정입니다.

소스 다운로드 (github)

2. 옵션표

옵션명 옵션값 타입 옵션 설명 기본값
initDate string 기본 value값을 집어넣을 수 있습니다.
반드시 문자열로 입력해야 합니다.
ex) '20190305'		 null
format string value값의 노출 포맷을 지정할 수 있습니다.
ex) 'yy.mm.dd', 'mm/dd/yyyy' 등등		 'yyyy-mm-dd'
visible boolean datepicker의 고정 여부를 정합니다. false
disableWeekends boolean 주말의 활성화 여부를 정합니다. false
text string 각종텍스트를 재정의 합니다.
데모 보러가기		 js파일 참고
periodFrom,
periodTo		 selector 기간설정 달력 사용 시 두개의 kronos를 연결할 때 사용 false
date array 공휴일 및 비활성화 날짜 지정
데모 보러가기		 disabled: [],
holiday: []
